# Tappy Bird

A Flappy Bird-inspired game created with LÖVE (Lua).

How to Play

- Press Space to flap.
- Avoid the pipes.
- Earn points by passing through gaps.
- Try to achieve the highest score and unlock medals.

Running the Game

1. Download "TappyBird.zip".
2. Extract the ZIP file.
3. Run "TappyBird.exe".

Features

- Sound effects and background music
- Pause and resume
- Countdown system
- Bronze, Silver, Gold, and Diamond medals
